### ✉Commit Convention
> 한 문제당 1 커밋을 준수합시다!<br>
> 커밋 형식을 꼭 지킵시다!

아래 링크를 참고해서 message template 설정 후 commit 작성  
해당 내용에서 config시에 해당 로컬 레포 루트 위치에서 아래 명령어로 해도 됨  
```
git config commit.template .gitmessage.txt
```
본문의 내용은 자유이며 작성 또한 자유이다. 단 리뷰 때는 설명 가능해야함
